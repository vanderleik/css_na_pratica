Color

Ao usar a propriedade "color", todo o texto será alterado. Ou seja todo o texto assumirá a cor selecionada no arquivo estilo.css.

Font

Usamos "font-family" para alterar a fonte do parágrafo. Se o navegador do usuário suportar essa fonte, então o texto será convertida para ela.

Fontes web Safe: Courier New, Georgia, Times New Roman, Trebuchet MS e Verdana.

Transformações e decorações

Usamos a propriedade "text-transform" para aplicar transformações no texto.

No caso do parágrafo acima, usei o "capitalize" para deixar a primeira letra de cada palavra em maiúscula. No caso deste parágrafo, usei o "uppercase" para deixar todo o texto maiúsculo.

No parágrafo acima use o "text-decoration: underline" para deixar todo o texto sublinhado.
